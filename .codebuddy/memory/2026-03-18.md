## 2026-03-18

- 为用户生成了 ESP32-S3 + ST7789(8pin) + 3按钮 + 蜂鸣器的完整接线和烧录指南
- 项目 Beans_PetGame (JurassicLife) 使用 `DISPLAY_PROFILE 3` 配置对应 ESP32-S3 + ST7789 240x240
- 引脚定义在 `board_esp32s3_st7789.hpp` 中：屏幕 SPI (GPIO 9-14)、按钮 (GPIO 4/5/6)、蜂鸣器 (GPIO 21)
- 项目使用 LovyanGFX 作为屏幕驱动库（非 TFT_eSPI），使用 Arduino IDE 编译
- 用户提供了已跑通的 TFT_eSPI 屏幕接线配置，据此修改了 LovyanGFX 驱动引脚
- 屏幕引脚变更: SCLK=18, MOSI=17, MISO=6, DC=15, CS=16, RST=4, BL=14
- 按钮因 GPIO 4/6 被屏幕占用，改为: LEFT=7, RIGHT=5, OK=8
- 蜂鸣器 GPIO 21 和 SD 卡引脚不变
