# 2026-03-18 工作日志

## ESP32-S3 + ST7789 240x240 适配改造

完成了 JurassicLife 恐龙电子宠物项目对 ESP32-S3-N16R8 + 1.54" ST7789 240×240 SPI IPS 屏幕的适配：

### 修改的文件
1. **`arduino/JurassicLifeFR/JurassicLifeFR.ino`** — 主代码文件
   - 新增 `DISPLAY_PROFILE_ESP32S3_ST7789` (值=4) profile
   - 添加了 ESP32-S3 的输入引脚 (BTN 4/5/6)、SD卡引脚 (36/37/35/38)、音频引脚 (GPIO 21) 配置分支
   - 添加了无触摸的 `readTouchScreen` 桩函数
   - setup() 中添加 ESP32-S3 初始化分支（背光PWM，跳过触摸校准）
   - 为方形 240x240 屏幕设置 ROT=0, FLIP_X/Y=false
   - SD 卡使用 SPI3_HOST 替代 HSPI
   - 全局添加 `#define LGFX_USE_V1`

2. **`arduino/JurassicLifeFR/board_esp32s3_st7789.hpp`** — 新建的板级 LovyanGFX 配置文件
   - LGFX_ESP32S3_ST7789 类：Panel_ST7789 + Bus_SPI (SPI2_HOST) + Light_PWM
   - 40MHz SPI 写入频率，invert=true（240x240 ST7789 需要反色）
   - 所有引脚通过 #define 宏集中管理

3. **`arduino/JurassicLifeFR/README.md`** — 更新的配置说明
   - 新增第10节 ESP32-S3 完整接线指南和 Arduino IDE 设置
